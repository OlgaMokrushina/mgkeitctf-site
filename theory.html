0. Введение в CTF

Capture The Flag — соревнования по кибербезопасности. Цель — найти флаги формата MGKEITCTF{...} в коде, файлах или трафике.

  • Форматы: Jeopardy, Attack-Defence, Mixed
  • Категории: Crypto, Stego, Web, OSINT, Forensics, Reverse, Misc
  • Советы: читайте условия, ищите подсказки, фиксируйте шаги решения

Инструменты для старта: CyberChef, VS Code, Wireshark, Binwalk, Python.

1. Стеганография

Скрытие информации внутри других данных: изображения, аудио, видео, архивы.

  • LSB в изображениях (младшие биты пикселей).
  • Метаданные EXIF (автор, координаты, камера).
  • Скрытые ZIP внутри PNG/JPG, составные файлы.
  • Аудиоспектр: текст в визуализации звука.

Инструменты: exiftool, steghide, zsteg, binwalk, Sonic Visualiser.

2. Форензика

Поиск и восстановление артефактов: трафик, образы, логи, дампы памяти.

  • Анализ PCAP (Wireshark), извлечение файлов из трафика.
  • Поиск строк и флагов: strings file | grep MGKEITCTF
  • Восстановление удалённого, временные линии событий.

Инструменты: Wireshark, Autopsy, Volatility, strings, foremost.

3. Криптография

Шифры и кодировки: от простых замен до RSA. Думайте про ключ, алфавит, таблицы.

  • Цезарь/ROT/Атбаш, Виженер, XOR (повторяющийся ключ).
  • Base64 — это кодирование, а не шифр.
  • Хэши: MD5/SHA; радужные таблицы, словари.
  • RSA: модули, экспоненты, уязвимые параметры.

Инструменты: CyberChef, dcode.fr, hashcat, Python (base64, binascii).

4. OSINT

Разведка по открытым источникам: поиск по сайтам, фото, профилям и инфраструктуре.

  • Google Dorking: site:example.com filetype:pdf
  • Метаданные EXIF (GPS, автор, устройство).
  • Поиск по картинке (Google, Yandex, TinEye), Shodan, Censys.

Инструменты: Exiftool, Shodan, Maltego, SpiderFoot, theHarvester.

5. Веб-технологии

Уязвимости сайтов и приёмы анализа HTTP/JS.

  • Просмотр исходника (Ctrl+U), DevTools, Network.
  • SQL-инъекции, XSS, cookies, CORS, заголовки.
  • robots.txt, sitemap.xml, скрытые эндпоинты API.

Инструменты: Burp Suite, curl, Postman, OWASP ZAP.

6. Программирование

Автоматизация решения задач и парсинг данных.

  • Python: requests, re, base64, itertools.
  • Bash: конвейеры, grep/sed/awk, jq для JSON.
  • Скрипты для перебора ключей/шифров.

Инструменты: Python3, bash, jq, Node.js (по желанию).

7. Реверс-инжиниринг

Понимание логики программ: как и где проверяется флаг.

  • Поиск строк и функций проверки (strings, Ghidra/IDA).
  • Дизассемблирование, декомпиляция, патчинг.
  • ELF/PE, .NET (dnSpy), Python .pyc (uncompyle).

Инструменты: Ghidra, IDA Free, radare2, dnSpy.
